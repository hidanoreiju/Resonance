# RESONANCE - 追想と再会の響き -

ブラウザでプレイ可能なダークファンタジーRPGです。
霧深い未練国の道を歩み、記憶を共鳴させながら門を目指します。

## 概要
- **ジャンル**: 探索型RPG
- **プレイ環境**: Webブラウザ
- **開発ツール**: HTML5, CSS3, JavaScript

## 操作方法
- **移動**: 各ボタン（▲, ▼, ◀, ▶）をクリックしてキャラクターを操作します。
- **戦闘**: 騎士ルーカスとの対峙時は、動きを見切り、隙を突いて勝利してください。
- **状態**: Resonance（共鳴値）が上昇すると、画面に異変が生じます。

## 機能
- 動的なマップ生成システム
- Resonance値に応じたUI変化と演出
- 戦闘・分岐選択のイベント処理
